4.6.4

Release Date: July 21, 2009

Bug Fixes / Minor enhancement

  • [01885] A large string value might prevent to display table grid (javascript error on display).

  • [01903] Dimension link by join table reference the instance from which it has been initialized. It cause severe error when working with instance duplication.

  • [01906] In some cases when triggers are disabled while inserting values into a table, the auto-increment value does not take into account the maximum value in the table.

  • [01908] Table grid header may not be sized as expected.

  • [01914] DataServices 'getChanges' operation should specifiy maxOccurs=unbounded on element 'updates/changes/change'.

  • [01915] DataServices 'getChanges' operation should specifiy maxOccurs=unbounded on element 'updates/changes/change/path'.

  • [01916] DataServices 'getChanges' operation should specifiy maxOccurs=unbounded on element 'deleted/predicate'.

  • [01921] Editor of multi-occurrences node does not show ongoing updates (occurrence deletion) if the form is redisplayed due to a form validation error.
