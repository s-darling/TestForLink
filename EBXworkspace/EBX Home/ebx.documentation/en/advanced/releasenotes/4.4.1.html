4.4.1

Release Date: April 15, 2008

In this version

  • Bug Fixes / Minor enhancement
  • Data Services

Bug Fixes / Minor enhancement

  • [01701] Multi-occurrence selection using a dropdownbox (html SELECT) is not large enough.

  • [01702] Time default is 00:00:00 for xs:time and xs:datetime field in EBX.

  • [01703] The purge of homes is not executed in an order that ensures integrity of homes hierarchy (a home record can be deleted but it still has children). If application server is abruptly shut down during a purge, it can produce errors when homes hierarchy is reloaded at restart.

  • [01704] XML Schema facet (minLength, pattern, etc) on osd:password are checked upon encrypted value.

Data Services

The Soap requests have been enhanced so as to be able to declare a trackingInfo field in the HEADER.
