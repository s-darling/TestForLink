4.6.2

Release Date: May 11, 2009

Bug Fixes / Minor enhancement

  • [01859] Perspectives are no longer modifiable.

  • [01860] DataModel Assistant basic types table must be Read-Only.

  • [01863] In DataModel Assistant, the tableRef constraint with an external container displays the adaptations contained in the DMA branch.

  • [01864] In DataModel Assistant, a NullPointerException is raised during the reverse procedure when multiple named complex nodes are defined in a schema.

  • [01866] Branch information access causes an IllegalArgumentException if branch key contains "|" (pipe) characters.

  • [01872]SessionPermissions.getNodeAccessPermission(SchemaNode, Adaptation) may return AccessPermission.getReadWrite() even if the node is set as read-only in the schema (osd:access='R-').

  • [01873] A read-only node (osd:access='R-', ValueFunction, ...) is editable in tabular view.

  • [01874] The service of import in table reports a 'success' even if the input file does not contain data that concerns the target table.

  • [01875] A NullPointerException occurs when computing a hierarchy dimension in the case of a recursive osd:select .
