New features

Documentation Search

Search in the documentation is henceforth available, on the upper-right corner of the window.

Relational mode support for Oracle 10g Release 2

Oracle 10g Release 2 has been validated as a supported persistence back-end for relational mode.

Positioning in ordered hierarchies

In a hierarchy, when an ordering field has been set, a newly created member is positioned last.

Default file name on export

A default filename is set for CSV and XML export, with the corresponding extension (.csv / .xml), in order to simplify the usage (ex: open file instead of download file via the user browser).

Version upgrade

How to upgrade an installed environment

In order to upgrade your version of EBX5, perform the following steps:

  1. Backup your repository, including the RDBMS and the directory ebx.repository.directory.

  2. Replace the *.war and ebx.jar files on your application server.

Bug Fixes

Core Cache

  • [4629] An exception occurs while restarting EBX without a server shutdown.

    A SOAP exception occurs on WebLogic 10.3.2 when restarting EBX without a server shutdown.

  • [4652] Error if primary key evolves and data set is deleted.

    If a dataset is created, then deleted, then the definition of a table primary key evolves, and finally a data set with the same reference and data model is recreated, an error occurs.

Data Modeling

  • [4631] Validation rule wizard can display mis-aligned icons.

    When entering wrong validation rules, the wizard wrongly aligns the icons.

Data Validation

  • [4633] In predicate or validation rule editor, an empty criterion field may lead to a NullPointerException.

    When the user inputs an empty value in the criterion text field on a validation rule editor, a NullPointerException is thrown during execution of

    the rule.

  • [4654] Min occurs cardinality constraint is disabled when the history is activated.

    The validation report ignores the min occurs cardinality constraint on fields inside a table (in semantic mode) when history mode is activated for the table.

Data Views

  • [4630] Data Hierarchies makes no sense on Data Spaces permissions.

    Data Hierarchies should not be available on Data Spaces permissions.

Data Workflow

  • [4649] After importing/exporting an archive in workflow modeling, the interface offers no way to quit the feature.

    When importing or exporting an archive in workflow modeling, the current workflow model is lost and no back button is displayed.

Documentation

  • [4628] If two HTTPRequest are executed at the same time, they can share the same session.

    Due to a side effect of the management of the HTTP session's identifier in some browsers (like Mozilla and Chrome), the cookies management should be deactivated for ebx web application in the configuration of the web server.

    Documentation has been modified accordingly.

Import Export

  • [4619] An exception occurs while importing an XML file into a table with a selection node.

    If the method ProcedureContext.setAllPrivileges is called (from a table trigger, a UI service, ... ) when importing the XML file, an exception occurs.

Miscellaneous

  • [4657] Exception logging may raise a NullPointerException.

    When the software throws an exception, the logging of this exception may in turn throw a NullPointerException, and will not report the original exception.

User Interface

  • [4557] Cannot add a criterion in the computation rule editor.

    An error message is displayed when a user wants to add a criterion to a condition under an 'IfTrue' or 'ifFalse' instruction.

  • [4622] In IE7, the paging component does not work.

    The paging component does not work when displayed in an enumeration selector.

  • [4646] Quotes are missing in the predicate editor for strings.

    In order to conform to the XPath standard, strings must be enclosed between quotes.

  • [4658] Dataspace action menu proposes 'View or edit' even if the user has read-only permissions.

    A user with read-only permission should see 'view content' label in dataspace action menu.

Users & Permissions

  • [4637] An exception occurs when importing into the data spaces permissions table.

    An exception occurs when importing into the data spaces permissions table. This was possible with 5.1.2.
