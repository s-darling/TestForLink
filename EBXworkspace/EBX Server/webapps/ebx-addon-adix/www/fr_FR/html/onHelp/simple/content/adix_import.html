Import des données depuis un fichier CSV ou Excel

Le processus pour importer les données vers les tables dans EBX5 depuis un fichier CSV ou Excel consiste en les étapes suivantes:

  • Téléchargement du fichier

  • Configuration de l'import

  • Liaison entre la feuille de calcul et les champs dans le modèle de données

  • Simulation et exécution de l'import

Initiez l'import du fichier en naviguant vers la table où les données seront importés en passant par son jeu de données dans la section Data. Puis, sélectionnez l'option import pour le type de votre fichier depuis le menu Services de l'espace de travail.

Options spécifiques aux fichiers CSV

Encodage du fichier

L'encodage des caractères du fichier importé

Séparateur

Le caractère utilisé pour séparer les cellules dans la feuille de calcul. Par défaut, les fichiers CSV utilisent les ; points-virgules.

Délimiteur

Le caractère utilisé pour marquer le début et la fin de contenu d'une cellule dans la feuille de calcul. Par défaut, les fichiers CSV utilisent les " guillemets.

Symbole décimal

Le caractère utilisé pour marquer un décimal

Formulaire pour correspondance des structures des données

Il faut définir la correspondance entre la structure de la feuille de calcul source envers la structure de la table dans laquelle vous importez les données.

Pour chaque champ dans la table, vous devez sélectionner une colonne du fichier téléchargé. Si la feuille de calcul importée contient une ligne de l'en-tête, l'import de l'add-on pour Microsoft® Excel fait une correspondance par défaut automatiquement si les noms des champs correspondent aux en-têtes des colonnes dans la feuille de calcul.

Dans cette page, vous pouvez également sauvegarder les options de configuration et de correspondance de l'import actuel. Pour utiliser la même configuration et la même correspondance dans un autre import, vous pouvez réutiliser cette préférence sauvegardée sur la page de téléchargement du fichier en utilisant l'option 'Chargement des préférences'.

Simulation et exécution de l'import

Avant d'exécuter réellement l'import, vous pouvez d'abord simuler le processus afin de détecter en avance si des problèmes se produiront pendant l'import. Cette vérification détecte les incohérences entre les cellules dans la feuille de calcul et les types de données définis dans le modèle des données de la table cible.

Vous pouvez décider de procéder à l'import, même s'il y a des erreurs de validation, en sélectionnant l'option 'Importer uniquement les enregistrements valides'. S'il existe des erreurs de validation et que vous continuez sans cocher l'option 'Importer uniquement les enregistrements valides', l'import échouera et aucun enregistrement ne sera importé.
