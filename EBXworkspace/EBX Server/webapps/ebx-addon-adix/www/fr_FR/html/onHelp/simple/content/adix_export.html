Export des données vers un fichier CSV ou Excel

Pour exporter des données depuis une table dans EBX5 vers un fichier CSV ou Excel, naviguez vers une table dans son jeu de données dans la section Data. Si vous voulez exporter les données de la table avec une vue, appliquez la vue sur la table. Puis, sélectionnez l'option d'export pour le type de fichier concerné depuis le menu Services dans l'espace de travail.

Note

L'export de l'add-on pour Microsoft® Excel est disponible uniquement pour les vues tabulaires simples. L'add-on pour Microsoft® Excel ne supporte pas l'export des données depuis les vues tabulaires multi tables et les vues hiérarchiques.

Configuration de l'export

Dans la formulaire de l'export, vous devez configurer les options d'export.

Options spécifiques aux fichiers CSV

Encodage de fichier

L'encodage du fichier exporté

Séparateur

Le caractère utilisé pour séparer les cellules dans la feuille de calcul. Par défaut, les fichiers CSV utilisent les point-virgules ;.

Délimiteur

Le caractère utilisé pour marquer le début et la fin des contenus d'une cellule dans la feuille de calcul. Par défaut, les fichiers utilisent les guillemets ".
